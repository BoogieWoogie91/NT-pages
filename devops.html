О продукте

DevOps — метод разработки программного обеспечения,
который синхронизирует работу разработчиков и
IT-специалистов. С DevOps можно быстро и надежно создавать,
тестировать и выпускать программные продукты и сервисы.

Для чего?

Суть метода — построить четко отстроенное и активное взаимодействие
между программистами, тестировщиками и системными администраторами.

Такая работа улучшает доверие между командами, повышает скорость
выпуска релизов программного обеспечения, позволяет своевременно
устранять ошибки и оперативно решать внештатные задачи.

Наши инструменты

NT corp использует современные и продвинутые инструменты, которые
востребованы на мировом рынке. Это значит, что ваша компания будет
обеспечена трендовыми методиками, которые улучшат работу ваших продуктов.

  • Управление исходными кодами Gitlab, Github

  • Непрерывная интеграция Gitlab CI

  • Контейнеры: Docker

  • IaC (infrastructure as Code): описываем инфраструктуру на Anisible и Shell, автоматически собираем образы в Docker

  • Экосистема (PaaS): Kubernetes

  • Система логирования: Elasticsearch + Kibana

  • Система мониторинга: Zabbix, Prometheus, Grafana

  • Виртуализация: KVM
