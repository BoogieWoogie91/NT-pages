Наши услуги

  • Aудит СУБД Oracle и OS сервера СУБД с предоставлением заказчику детального отчета

  • Весь спектр управления базами данных: создание, настройка, изменение, удаление, модификация и т.д

  • Обслуживание баз данных, базирующихся на решениях высокой доступности и отказоустойчивости: кластеризация, зеркалирование, репликация и т.д

  • Оперативное решение вопросов, связанных с работой баз данных

  • Мониторинг основных показателей баз данных: доступность, исправность, объем и свободное пространство

  • Управление доступом к базам данных и обеспечение их безопасности

  • Резервное копирование и восстановление баз данных

  • Разработка предложений по улучшению производительности баз данных

  • Мониторинг событий, возникающих в процессе работы баз данных

  • Кроме СУБД Oracle у нас есть опыт администрирования таких СУБД как: PostgreSQL, OpenEdge (Progress), MySQL
